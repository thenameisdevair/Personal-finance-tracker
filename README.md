# Personal Finance Tracker

A lightweight, vanilla-JavaScript app to track income & expenses with localStorage persistence. Includes custom categories, sorting & filtering, CSV export, and an Income vs Expense chart (Chart.js). Built for **Web3Bridge Cohort XIII** pre-qualification.

## Demo

GitHub Pages: (add your live URL after enabling Pages)

## Features

- Add **income/expense** with amount, date, category, notes
- **Custom categories** per type
- **Sort & filter** by date, type, category, amount, and notes text
- **CSV export** of the current (filtered) view
- **Chart**: Income vs Expense (Chart.js CDN)
- **localStorage** persistence
- **Responsive** design

## Quick Start

Open `index.html` in a browser.

## Tech

Vanilla JS, HTML, CSS. Chart.js via CDN. Data stays in the browser.
